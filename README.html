Reference for the local grant research and application system in 11_Grants/.

File Naming Standard

Use YYYY-MM-DD_ClientOrGrant_DocumentType_Context_v01.ext.

  • 2026-05-16_SBDC_Proposal_Draft_v01.docx
  • 2026-05-22_SBDC_Budget_Justification_v02.xlsx
  • 2026-06-05_SBDC_Agreement_v03_Executed.pdf

California Grants Portal Workflow

Portal: grants.ca.gov

  1. Search for relevant state opportunities.
  2. Save official guidelines in Source_Documents/.
  3. Create one note per opportunity in Open_Opportunities/.
  4. Move records to Submitted/, Awarded/, or Closed/ as status changes.

Grant Writing Software

Tool Type Best Use Examples Notes
All-in-one platform Discovery, tracking, and drafting in one system Instrumentl, Grantable Most useful when prospecting volume or deadline load becomes difficult to manage manually.
Writing assistant Drafting first-pass narratives and revisions Grantable, ChatGPT Best used with local templates and human review for compliance and voice.
Research database Private and foundation prospecting Candid FDO, GrantStation Useful when state and federal portals are not enough.
Lightweight local stack Low-cost workflow California Grants Portal, tracker, local templates Good default for a solo operator until clear bottlenecks appear.

Use one paid platform at a time when testing. Choose for the bottleneck you actually have: prospecting, drafting, or foundation research.

Daily Use

  1. Review opportunities in seeking/ and the California state folders.
  2. Log promising grants in the Excel tracker.
  3. Track fit, deadlines, and requirements in the opportunity template.
  4. Move serious prospects into an active grant work folder when ready to apply.
