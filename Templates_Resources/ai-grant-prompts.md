# AI Grant Prompts

Use these prompts with saved source material, local templates, and client facts. Treat outputs as drafts, not final submissions.

## 1. Opportunity Fit Check

```text
Review the following grant opportunity and tell me whether it is a strong fit.

Client summary:
[paste client summary]

Grant summary:
[paste grant summary or NOFO excerpt]

Return:
1. Fit score from 1-10
2. Top reasons it fits
3. Top risks or disqualifiers
4. Required attachments or partners
5. Recommendation: pursue, monitor, or decline
```

## 2. Proposal Outline

```text
Using the grant requirements below, build a proposal outline with section headings, required evidence, and a short note on what content belongs in each section.

Grant requirements:
[paste requirements]

Client context:
[paste client background]
```

## 3. Draft Narrative From Notes

```text
Write a first draft for the following grant section using the notes provided.
Keep the tone professional, specific, and plain.
Do not invent facts. If information is missing, add a short [MISSING INFO] marker.

Section name:
[paste section]

Notes:
[paste bullets, metrics, and source text]
```

## 4. Compliance Check

```text
Compare this draft against the requirements and list:
1. missing elements
2. unsupported claims
3. places that are too vague
4. word-count or formatting risks

Requirements:
[paste requirements]

Draft:
[paste draft]
```

## 5. Budget Justification Support

```text
Turn these budget line items into a concise budget justification.
Explain why each cost is necessary for project delivery and keep the language grant-appropriate.
Do not add costs that are not listed.

Budget lines:
[paste budget lines]

Project summary:
[paste project summary]
```

## 6. Executive Summary Rewrite

```text
Rewrite this draft into a tighter executive summary for a grant reviewer.
Keep it clear, concrete, and persuasive.
Limit jargon and keep all original facts.

Draft:
[paste draft]
```

## 7. Post-Draft Review

```text
Act as a strict grant reviewer.
Score this draft on clarity, alignment, specificity, feasibility, and evidence.
Then give the five highest-impact revisions.

Draft:
[paste draft]

Opportunity summary:
[paste summary]
```
