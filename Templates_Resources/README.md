# Templates and Resources

This folder stores reusable grant-writing materials for drafting, review, and submission prep.

## Current Files

- `ai-grant-prompts.md`: reusable prompts for fit checks, drafting, compliance review, and revision.
- `proposal-outline-template.md`: base outline for organizing a new proposal before drafting.

## Suggested Use

1. Start with `proposal-outline-template.md` when a grant is worth pursuing.
2. Use `ai-grant-prompts.md` to turn notes and source documents into draft content.
3. Save grant-specific working files in the relevant opportunity or client folder, not in this shared templates folder.

## Naming

For new reusable files, keep names short and functional, for example:

- `budget-justification-template.md`
- `letters-of-support-checklist.md`
- `post-award-report-template.md`
