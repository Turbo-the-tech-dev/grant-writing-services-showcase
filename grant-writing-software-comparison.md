# Grant Writing Software Comparison

This comparison is for workflow planning in this repository. It is not a live pricing sheet. Verify current features and pricing before buying.

## Recommended Approach

Start with the local stack already in this workspace:

- `California Grants Portal`
- `Research_Database/`
- `2026_Grant_Opportunity_Tracker.xlsx`
- local templates and prompt files
- `ChatGPT` for drafting and review

Add paid tools only when a specific bottleneck is recurring.

## Tool Comparison

| Tool | Primary Strength | Best For | Tradeoff | Suggested Use Here |
| --- | --- | --- | --- | --- |
| Instrumentl | Opportunity discovery and tracking | High prospecting volume, multi-client pipelines | Can overlap with manual trackers if volume is low | Test if finding and tracking grants is taking too much time |
| Grantable | Drafting and grant workflow support | Faster first drafts and revision support | Requires careful human review for compliance and tone | Use with local templates in `Templates_Resources/` |
| ChatGPT | Flexible drafting, outlining, editing | Narrative development, checklists, gap analysis | Needs strong prompts and source discipline | Keep as the default writing assistant |
| Candid FDO | Foundation research depth | Private foundation prospecting | Most useful only if private funders matter | Add when expanding beyond public grants |
| GrantStation | Lower-cost research support | Broader prospecting on a budget | Usually less central than a core writing tool | Consider as a budget research layer |
| Local-only workflow | Lowest cost and highest control | Solo operators, early-stage systems | More manual effort | Strong default for this repository |

## Selection Rules

- If prospecting is the problem, test discovery tools first.
- If writing speed is the problem, test drafting tools first.
- If private foundations are the target, add a research database.
- Avoid paying for two tools that solve the same bottleneck.
