# Local Grant Services Website

This folder contains the public-facing website concept and implementation for a local grant writing consultancy.

## Files

- `site/index.html` contains the static one-page website.
- `site/styles.css` contains responsive styling and brand visuals.
- `site/app.js` contains small interaction helpers.
- `website-development-plan.md` preserves the full build plan.
- `docs/future-backlog.md` tracks future enhancements.
- `docs/content-checklist.md` tracks launch content requirements.

## Local Preview

Open `site/index.html` in a browser. No build step is required.

## Positioning

The website is written for nonprofits, small businesses, schools, municipalities, and startups that need practical grant strategy, application management, and reporting support.
