# California State Grant Research

Use this folder to track opportunities identified through the California Grants Portal and agency-specific state funding pages.

## Subfolders

- `Open_Opportunities/`: active prospects under review.
- `Source_Documents/`: saved NOFOs, guidelines, and source PDFs.
- `Submitted/`: applications already filed.
- `Awarded/`: funded opportunities and award follow-up.
- `Closed/`: expired, declined, or no-go opportunities.

## File Naming

Use:

`YYYY-MM-DD_CA_ProgramName_DocumentType_v01.ext`

Examples:

- `2026-05-16_CA_Climate_Grant_Opportunity_Notes_v01.md`
- `2026-05-16_CA_Climate_Grant_Guidelines.pdf`
